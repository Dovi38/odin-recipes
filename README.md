# odin-recipes

'Odin-recipes' project is based on plain Html to create basic recipe website.
Website consist of a main index.html page which have links to a few recipes.
Each recipe page is built using html tags:
headings tags <h1>...<h6>,
links <a> ,
image <img>,
paragraph <p>,
ordered list <ol>,
unordered list <ul>,
list items <li>
